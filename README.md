# AI BUG 记录工具

基于 LLM（大语言模型）的智能 BUG 记录助手，支持 AI 对话式信息收集、语义搜索历史相似 BUG、自动生成标准化 BUG 文档。

## 📁 项目结构

```
AIagent/
├── main.py              # GUI 程序入口（Tkinter 实现）
├── bug_record_core.py   # BUG 记录核心逻辑
├── llm.py               # LLM AI 调用封装
├── ui.py                # UI 抽象接口
├── bug_template.md      # BUG 文档模板
└── bug_records/         # BUG 记录存储目录
    ├── bugs_index.json  # BUG 索引文件
    └── BUG_*.md         # BUG 记录文件
```

## 📝 模块说明

### 1. `ui.py` - UI 抽象层

定义统一的 UI 接口，支持终端和 GUI 两种实现：

```python
class UI(ABC):
    @abstractmethod
    def write(self, text): pass
    
    @abstractmethod
    def read(self, prompt="") -> str: pass
    
    @abstractmethod
    def log(self, text): pass

class TerminalUI(UI): 
    """终端实现，使用 print/input"""
    def write(self, text): print(text)
    def read(self, prompt=""): return input(prompt)
    def log(self, text, end="\n"): print(text, end=end)
```

### 2. `bug_record_core.py` - 业务逻辑层

核心类说明：

| 类名 | 说明 |
|------|------|
| `BugToolUI` | UI 适配器，继承自 `UI` 抽象类 |
| `BugDialogTool` | BUG 对话框工具，处理用户交互和数据收集 |
| `BugRecord` | 程序入口封装，处理主菜单逻辑 |

**核心方法：**

| 方法 | 说明 |
|------|------|
| `start_conversational_record()` | 启动对话式 BUG 记录 |
| `handle_bug_description()` | 处理 BUG 描述输入 |
| `_ai_extract_all_fields()` | AI 自动提取字段 |
| `_search_similar_bugs_ai()` | AI 语义搜索相似 BUG |
| `_save_bug_record()` | 保存 Markdown 格式 BUG 记录 |

### 3. `llm.py` - AI 服务层

提供 AI 调用封装，包含超时、重试、解析等机制：

| 函数 | 说明 |
|------|------|
| `ai_chat(user_input)` | AI 闲聊对话 |
| `ai_extract_all_fields(input, data, fields)` | AI 全字段提取 |
| `ai_generate_question(field, name, data, desc)` | AI 生成自然提问 |
| `ai_intent_recognize(input, options, scene)` | AI 意图识别 |
| `search_similar_bugs(index, query, ...)` | AI 语义搜索 |

## 🎯 核心特性

### 1. AI 智能字段提取
- 用户输入任何内容，AI 自动匹配所有字段
- 支持多行输入，一次性填写多个字段
- 支持字段编号格式：`1: 内容`

### 2. AI 语义搜索
- 输入 BUG 描述，自动匹配历史相似记录
- 生成自然的相似原因描述
- 给出相似度百分比

### 3. AI 意图识别
- 支持中英文模糊匹配
- 同义词识别（新建/创建/新增 → 同一操作）
- 关键词优先匹配策略

### 4. 标准化输出
- 自动生成符合模板的 Markdown 文件
- 包含完整的索引管理
- 支持查看、搜索、列表等操作

## ⚙️ 配置说明

### 环境变量

在 `.env` 文件中配置：

```bash
# API 密钥（必填）
export LLM_API_KEY="your_api_key_here"

# API 地址（可选，不设置则使用默认值）
export LLM_API_URL="user_url"

# 模型名称（可选）
# export LLM_MODEL="qwen3.5-plus"
```

### LLM 配置 (`llm.py`)

```python
class LLMConfig:
    LLM_TIMEOUT = 100           # 超时时间（秒）
    MAX_RETRY_COUNT = 2         # 最大重试次数
    REQUEST_TIMEOUT = 60        # 请求超时（秒）
    DEFAULT_MODEL = "qwen3.5-plus"
    
    @classmethod
    def get_default_api_url(cls):
        """从环境变量 LLM_API_URL 读取，未设置时使用默认值"""
        return os.environ.get("LLM_API_URL", "user_url")
```
